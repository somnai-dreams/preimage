Loading patterns

Runs the same gallery workload under each named loading pattern and times the phase milestones. streamed fires image fetches per probe resolve; skeleton-first finishes probing before any image fetches; manifest-hydrated skips probing entirely via provided aspect ratios; throttled caps parallel image fetches while probes run at full tilt; viewport-first gets first-screen skeletons in place, then current-viewport images, then throttled mounted work. All five drive the same virtual tile pool.
