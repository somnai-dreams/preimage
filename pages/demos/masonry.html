Masonry — reflow during image loads

Two grids, same photos, separate Run buttons so you feel each one. The left uses stock <img> tags with no reserved dimensions — the column stretches as each image decodes and every tile below it shifts. The right calls prepare() per image first, then commits a shortest-column layout in one synchronous pass; tiles are positioned before any pixel paints, and the dominant-color placeholder holds the spot.
