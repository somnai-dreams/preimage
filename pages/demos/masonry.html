Masonry — images without declared dimensions

Sixty photos, loaded side by side. Most images on the web arrive without width / height attributes: markdown doesn't ship them, CMSes omit them, third-party feeds and user uploads almost never have them. Left panel shows what that looks like today. Right panel calls prepare() on each image first, then uses the measured aspect ratios to place them into a stable masonry — the layout logic is a plain shortest-column fill in the demo itself, not part of preimage.
