Chat — inline images in a resizable bubble

A chat message with icons, stickers, and photo attachments mixed into the text. inlineImage turns each image into a pretext RichInlineItem whose extraWidth equals the measured image's rendered width — pretext treats it as an atomic pill that wraps with neighboring text and never breaks. Drag the slider to resize the bubble; the text re-breaks and images shuffle to their new slots in a single synchronous flow pass, no DOM measurements, no reflow.

Bubble width 420px
Source items: a sequence of text fragments and inlineImage(src, { font, height }) items, fed to prepareRichInline. On every resize we call walkRichInlineLineRanges — pure arithmetic over the cached text widths, no DOM reads. Image fragments carry the PreparedImage handle, so rendering them is a single <img src=…> with exact width/height pre-known.
