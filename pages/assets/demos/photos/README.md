# Demo photos

Midjourney-generated photos used by the reflow/sizing demos. Not part of the
published library — included only for the demo pages. Kept here rather than
pulled from a third-party CDN so demos are deterministic, offline-safe, have
honest cache-busting, and don't depend on redirect chains.

## Naming convention

Drop files in this directory named `01.png` through `NN.png` (zero-padded to
two digits). Order is arbitrary but stable — `photo-source.ts` enumerates the
list in numeric order.

## Expected files

34 photos — see `photos.json` (to be generated) for the manifest with natural
dimensions once the files land.
